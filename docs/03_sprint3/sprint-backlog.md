# スプリント3 バックログ

## スプリントゴール

**「フォームからTODOを作成でき、不正な入力は入力値を保持したまま差し戻され、
成功時はメッセージつきで一覧に戻る(リロードしても二重登録されない)」**

対応するユーザーストーリー: [US-3(TODOを作成する)](../00_project/user-stories.md)
対応するPBI: #8(作成フォーム+保存)、#9(バリデーション境界値と「わざと失敗」実験)

写経目安時間: **2.5〜3時間**

## このスプリントで初登場する概念(予告)

GETでデータを変えない原則 / CSRF / バリデーションと自動差し戻し / old値 /
PRGパターン / フラッシュデータ / ステータスコード302・405・419
