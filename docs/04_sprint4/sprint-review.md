### トラブル1: PR #5のマージが「Pull Request is not mergeable」で失敗

- **症状**: push直後の `gh pr merge` がGraphQLエラー
- **調査**: `gh pr view --json mergeable,mergeStateStatus` → 両方 **UNKNOWN**
- **原因**: push直後はGitHub側のマージ可否判定が未計算(コンフリクトではない)
- **解決**: 数秒待って再実行 → 正常にマージ
- **学び**: エラーメッセージの字面(not mergeable)だけで「コンフリクトだ」と
  思い込まない。まず実データ(mergeable=UNKNOWN)を見る — 調査の型は運用作業でも同じ
